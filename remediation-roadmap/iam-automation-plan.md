# IAM Remediation Roadmap

**Phase 1 – Immediate**
- Define 24-hour SLA
- Eliminate shared admin accounts
- Quarterly access review

**Phase 2 – Process Integration**
- HRIS → ITSM workflow trigger
- Centralized account inventory

**Phase 3 – Automation**
- IAM lifecycle management
- Auto-disable inactive accounts
- Privileged access certification
